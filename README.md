# Proyectos dbt: Jaffle Shop y LinkUp Raw Data

Este repositorio contiene dos proyectos dbt independientes para el manejo y transformación de datos en Snowflake.

## Proyectos

### 1. jaffle_shop
Un proyecto de demostración clásico de dbt que modela datos de un negocio minorista (e-commerce).
- **Propósito:** Demostrar conceptos básicos de dbt, como modelos, pruebas y materializaciones.
- **Modelos clave:** `customers`, `orders`.

### 2. linkup_raw_data
Un proyecto orientado a la ingestión y transformación de datos crudos (probablemente relacionados con LinkedIn o datos de empleo, basado en el nombre).
- **Estructura:** Sigue la arquitectura de capas (Staging -> Intermediate -> Marts).
- **Propósito:** Limpieza, estandarización y preparación de datos para análisis.
